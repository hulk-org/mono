The EGM Compendium (2025, Josh Harmon + Mollie L Patterson) lives at `private/universal/vaults/egm-page-vault/` — 368 pages, extracted as per-page PDF + Markdown.

**Design patterns that map to wrkstrm-games:**
- **Review Crew**: 4-5 simultaneous reviewers with distinct scores + short writeups → agent review panels for shipped builds
- **Series Speedruns**: franchise retrospectives with avg scores, best/worst → agent performance history timelines
- **Score grids**: scannable numeric comparison tables → HUD overlays for agent stats
- **Sidebar callouts**: ARCHIVE CODE, FACT FILE, WHATEVER HAPPENED TO → room info panels, file inspector
- **1000 Best Games list**: ranked completionist collection → achievement/collection tracking for explored directories
- **Sushi-X masked reviewer**: reviewer-as-persona → agent identity as game character
- **Reader letters / platform debates**: community voice → agent chat / task commentary

**Key insight:** EGM's lasting format is *scannable numeric systems with personality*. Scores are data; reviewer voice is narrative. The game UI should do both — show real bench numbers and test results through the lens of agent personas reviewing each other's work.
