コレツフ (Koretsufu) = the Collective. Decided 2026-04-11, after exploring コレクティブ (textbook), regional American variants, and グンチ (Gunchi, rejected).

**Why:** Four pure full-size unmodified katakana. No small-kana modifiers (ティ rejected as impure). Derived from American pronunciation of "collective" compressed to four syllables: "ko-re-tsu-fu." The ツ (tsu) is a percussive stop that also looks like a smiley face to Americans — the collective is winking from inside the name. The フ (fu) is the breathiest katakana consonant — the French "collectif" exhale ending.

**How to apply:** Use コレツフ (Koretsufu) as the canonical name for the Collective organism class. Romanized: Koretsufu.

The naming chain:
- タチコマ (Tachikoma) → コマ/Koma = bounded execution unit (4 katakana, constructed from kanji roots)
- コレツフ (Koretsufu) = collective / emergent swarm intelligence (4 katakana, compressed vocal transliteration)

Both are four pure full-size katakana characters. No modifiers. The pair:
```
タチコマ   ta-chi-ko-ma    — the machine
コレツフ   ko-re-tsu-fu    — the collective
```
