The metal-iso-bench evolves into the **Logikoma training ground** — the visual environment where Koma (Tachikoma workers) train before deployment.

**Connection to substrate:**
- Komo (short for Tachikoma) are bounded workers deployed by Tau via CLIDE
- Species presets: Ant/Hound/Fox/Spider/Owl — each should have a distinct silhouette on the iso grid
- The training ground is where you see them practicing tricks, learning expertise, running tasks
- Feeds into Agent RPG's studio view — the training ground IS a room in the studio

**Visual identity:**
- Isometric diamond grid floor (green = safe, red = hazard, blue = data)
- Koma as billboard sprites with ground shadows
- Training obstacles/stations as buildings on the grid
- Progress indicators per Koma (XP bars, skill dots)

**How to apply:** Rename metal-iso-bench to logikoma-training-ground. Connect to real Koma deployment data when issue 008 (real data binding) ships.
