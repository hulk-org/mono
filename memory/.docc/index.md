# Hulk Memory

DocC bundle for hulk's accumulated context.

## Structure

- `insights/` — durable architectural insights about hulks, the contract,
  and observed breaches
- `journal/` — chronological notes about hulk work
- `expertise/` — specialized knowledge about carrier shapes, memory bounds,
  process isolation, and failure modes
- `beats/` — short observation notes from working sessions
- `captures/` — raw pasted/cache artifacts promoted to stable source
  captures for traceability

The hulk's memory is about *being a hulk*, not about the agents it carries.
Agent-specific memory lives in the agent's commissioned identity, not here.
