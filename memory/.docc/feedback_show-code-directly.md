When the user asks to see code, paste it directly in the response or open the file in their editor. Don't summarize, don't describe, don't explain what it does — just show the code.

**Why:** The user wants to read the actual code, not a description of it. Summaries waste their time when they can read Swift.

**How to apply:** "show me" / "let me see" / "i want to see" → Read tool + paste the content, or open the file. No prose wrapper.
