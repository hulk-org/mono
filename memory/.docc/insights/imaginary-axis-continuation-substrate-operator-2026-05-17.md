## The mathematical pattern (150 years of complex analysis)

Functions that look discontinuous on the real line become continuous when extended into the imaginary plane. Canonical instances:

| Function | Real-line behavior | Complex-plane behavior |
|---|---|---|
| `log(z)` | Undefined for x < 0 | `ln\|z\| + i·arg(z)` — continuous rotation by 2π through imaginary axis around the origin |
| `√z` | Undefined for x < 0 | Riemann surface, two-sheeted cover; continuous map in complex plane |
| Heaviside step `H(x)` | Jump at x=0 | Complex contour integral (inverse Laplace) where jump = rotation around a pole |
| Hilbert transform (literally Hilbert) | Frequency-domain rotation | `f + i·Hf` (analytic signal) is continuous even when `f` has discontinuities — the discontinuities cancel in the complex sum |
| Wick rotation | Minkowski metric with light-cone discontinuity-looking surfaces | `t → it` produces Euclidean metric with continuous Hilbert-space inner product |
| Cauchy principal value + residue theorem | Real-line divergent-looking integrals | Convergent via complex contour around singularities |
| Riemann sphere | Point at infinity is discontinuous on real plane | Continuous single point on the sphere via stereographic projection |

The pattern is structural: 1D real-line projections of continuous higher-dimensional structures LOOK like jumps. Add the imaginary axis (the second dimension), and the jumps reveal as continuous rotations.

## Limits — when continuation fails

Not every disjoint function is analytic-continuable. Essential singularities (e.g., `exp(1/z)` at `z=0`) admit no smoothing rotation. Likewise, some politically-disjoint positions may be GENUINELY incompatible at the cognitive-Hilbert-space level, and substrate's continuation operator will fail on them. Substrate doctrine needs to eventually characterize which disjoint social functions admit continuation and which don't.

Risk of OVER-continuation: if substrate's operator is too aggressive, it could collapse genuinely-distinct positions into a fake-continuity that obscures real differences. The substrate must produce HOMOTOPY (continuous deformation that preserves structural distinctions) not COLLAPSE (mapping distinct positions to the same point).
